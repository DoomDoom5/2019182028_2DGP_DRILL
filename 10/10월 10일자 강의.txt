객체 : 실제 세계에 존재하는 물체
추상화 : 핵심 내용을 강조하기 위해 나머지를 간략히 보여주는 화법/기술

속성 : 객채의 상태 ex) x,y, width, height, state
속성 값 : 객체의 상태 값 ex) x = 50.0, state = RUNNING , height = 100.0kg


행위 :  객체의 상태를 바꿔주는 것

속성 + 행위 = 객체
 
클래스 : 객체들을 재사용 하기 위해 만들어진 
객체를 정의하는 소프트웨어 단위
객체를 찍어내는 '도장'


객체 == 인스턴스

생성자 : 객체의 처음모양을 만드는 함수
